重新定义 Windows 卸载体验。

BCUninstaller (简称 BCU) 是一款自由（言论自由意义上）的程序卸载工具。它擅长以最少的用户操作批量卸载大量应用程序。

它可以清理残留文件、检测孤立应用、根据预设列表运行卸载程序，还有更多功能。

尽管 BCU 是为 IT 专业人员设计的，但任何对 Windows 中应用程序安装/卸载有基本了解的人都可以使用。

采用 Apache 2.0 开源许可证，完全免费用于私人和商业用途。

// 03 — WORKFLOW

三步还原纯净系统。

从检测到清理，全流程可控。

BCU 不只是卸载器 —— 它是一套完整的应用生命周期管理工具。

  • 扫描所有已安装应用，按来源、占用空间、最后使用时间排序
  • 选择需要卸载的目标，配置静默参数与清理策略
  • 执行批量卸载，自动清理注册表、文件夹、计划任务等残留
  • 导出日志报告，便于审计与回溯
